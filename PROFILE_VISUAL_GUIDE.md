## Color Scheme

### Card Styling
- Background: Cosmic gradient (purple/blue)
- Border: Primary color with opacity
- Hover: Shadow with glow effect
- Border radius: 8px

### Stat Colors
- Blue: Lessons (completed)
- Green: Quizzes (taken)
- Yellow: Challenges (completed)
- Orange: Streak (days)
- Purple: Questions (answered)
- Emerald: XP (earned)
- Pink: Level (current)

### Activity Types
- Blue: Lesson completed
- Green: Quiz taken
- Yellow: Challenge completed
- Purple: Level up
- Orange: Streak milestone
- Pink: Achievement unlocked

### Accuracy Colors
- Green: â‰¥80%
- Yellow: 60-79%
- Red: <60%

### Heat Map Intensity
- Level 0: White/Muted (no activity)
- Level 1: Light green (1-2 activities)
- Level 2: Medium green (3-4 activities)
- Level 3: Dark green (5-6 activities)
- Level 4: Darkest green (7+ activities)

## Responsive Design

### Desktop (>1024px)
- 4 columns for stats
- Side-by-side charts
- Full heat map visible
- 3-column achievement grid

### Tablet (768-1024px)
- 2 columns for stats
- Stacked charts
- Scrollable heat map
- 2-column achievement grid

### Mobile (<768px)
- 1 column for all
- Full-width charts
- Scrollable heat map
- 1-column achievement grid
- Stacked navigation

## Interactive Elements

### Hover Effects
- Stat cards: Scale up + shadow
- Buttons: Color shift
- Chart points: Enlarge + tooltip
- Heat map cells: Ring outline
- Achievement cards: Scale up
- Timeline items: Subtle highlight

### Click Actions
- Tab navigation: Switch views
- Time range selector: Refetch data
- Category filter: Filter achievements
- Achievement card: Open detail modal
- View buttons: Navigate to learning
- Back button: Navigate to dashboard

### Animations
- Page load: Fade in
- Tabs: Smooth transition
- Charts: Animate on data load
- Cards: Staggered entrance
- Modals: Scale in

## Accessibility

### Keyboard Navigation
- Tab: Navigate through elements
- Enter/Space: Activate buttons
- Escape: Close modals
- Arrow keys: Navigate within components

### Screen Reader Support
- Semantic HTML structure
- ARIA labels on icons
- Alt text for images
- Descriptive link text

### Focus Indicators
- Visible focus rings
- High contrast focus
- Logical tab order
- Skip navigation option
